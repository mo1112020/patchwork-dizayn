Behaviour summary

  • Sidebar shows every category with a count badge — auto-built from your texture data
  • "Tümü" (All) is always the first item and selected by default
  • Clicking a category resets to page 1 and shows only that category's textures
  • 18 textures per page in a responsive grid (3 cols → 2 cols on narrow screens)
  • Pagination shows "Page X / Y · N textures" so you always know where you are
  • No backend changes needed — all filtering and paging happens in the browser with the already-loaded texture list
